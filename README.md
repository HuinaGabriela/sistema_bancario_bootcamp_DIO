# Sistema Bancário (POO) - Bootcamp DIO
Projeto desenvolvido como parte do Bootcamp da DIO com foco em Programação Orientada a Objetos (POO) utilizando Python.

## 🚀 Funcionalidades
- Criar usuário
- Criar conta
- Depositar
- Sacar
- Extrato

## 🧱 Estrutura
- models/
- transacoes/
- interface/
- utils/

## ▶️ Como executar
python -m src.main

ou pelo aquivo input.txt comando(windowns powershel):
Get-Content input.txt | python -m src.main

## 🧪 Testes
pytest --cov=src

## 🧠 Conceitos aplicados
- POO
- Herança
- Polimorfismo
- Encapsulamento
- ABS (Classes Abstratas)
